# xArm-C++-SDK
----------

## Overview
- Only support Linux


## Doc
- [xArm](doc/xarm_cplus_api.md)

### Install

Get the code:

```bash
git clone https://github.com/xArm-Developer/xArm-CPLUS-SDK.git
```

#### Linux
- Build (only in Linux):
  ```
  make clean && make
  ```

- Run Test (only in Linux and MacOS)
    ```bash
    ./build/test_get 192.168.1.192
    ```
